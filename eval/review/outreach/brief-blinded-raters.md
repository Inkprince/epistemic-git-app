# Invitation: rater for a blinded comparison study

*Draft outreach note for the blinded baseline study. Recipients need **no** domain expertise — the
point is careful, consistent scoring against a fixed rubric. Recruit at least three so we get real
inter-rater agreement (the study minimum is two). Personalize and fill brackets.*

---

Dear [name],

I'm running a small blinded evaluation for a Future of Life Foundation competition entry, and I'm
looking for a few careful people to score some anonymized outputs. **No special background is needed** —
if anything, it's better that you're not invested in the answer.

**The setup.** Three different AI research approaches were each given the same seven tasks across three
cases (a physics safety question, a COVID-origins sub-question, and a nutrition question). All of their
outputs have been stripped of anything identifying which system produced them, shuffled, and given
opaque labels. You won't be told which is which — that's the point.

**What you'd do.** For each task, read the (anonymized) output and answer a short set of factual,
rubric-based questions — for example: does a cited source actually support the sentence it's attached
to? Is a stated claim backed by the evidence given? Did the output correctly identify the one thing
that, if resolved, would most change the conclusion? You're scoring against a fixed checklist, not
giving an opinion, and there are no trick questions.

**Time.** About [60–90 minutes], done whenever suits you. You'll get a short instruction sheet
(`EVALUATOR.md`) and a packet of anonymized outputs. Everyone scores the same items independently so we
can measure how much raters agree.

**Ground rules.** Please don't try to guess or look up which system made which output, and don't compare
notes with other raters until everyone's done — independence is what makes the numbers meaningful.
[Compensation: fixed [amount] for your time, or: this is unpaid.]

Would you be up for it? If so, I'll send the packet and instructions.
